  • Design:
    • Contrast: The color scheme makes it easy to read and provides good contrast between the text, navigation, and footer.
    • Repetition: The font-family is suitable for the page. Headers are bold and makes it easy to understand what information the user is reading.
    • Alignment: The alignment of the page feels off-center and could benefit from padded margins. Having the navigation bar to the left of the page is a nice touch, however, it would be better to not scroll to the side to see the main content.
    • Proximity: It would be good to use less horizontal spacing between the nav bar and the main element. Vertical spacing is perfect.
  • Page:
    • Page contains header, main, and footer. Footer should not contain the itis3135 site links.
    • h1 is the name of the website and the h2 is the name of the page
    • The main element is sectioned well and portrays organization
    Requirements:
    • Has five functional pages
    • The home page is listed under index.html
    • The nav bar is functional
  • Conclusion
    • The FAQ portion of the site provides a drop down with provided information and makes for easy info look up.
    • The calendar portion is a nice touch and shows a good working knowledge of java.
    • Unfortunately, none of the images are working on the site and there are a few html errors on the schedule.html.
    • Overall the site is good but could use a few tweaks to make it more user-friendly.
